## 📊 Data Analytics Essentials

**Issued By:** Cisco Networking Academy  
**Completion Date:** June 20, 2026

### 📖 Course Overview

This course provided a comprehensive introduction to the data analytics lifecycle, covering data collection, preparation, visualization, statistical analysis, relational databases, SQL, Tableau, and ethical data practices. Through practical exercises and real-world scenarios, I developed the foundational skills required to transform raw data into meaningful insights for informed decision-making.
